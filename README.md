# 🚀 Git & GitHub Practice Project

A simple **HTML, CSS, and JavaScript** project created to practice the complete Git & GitHub workflow.

This repository focuses on learning version control rather than web development. Throughout this project, I practiced creating commits, tracking file changes, working with branches, merging code, and managing a GitHub repository.

---

## 📌 Project Overview

This is a simple Profile Card application that displays:

- Name
- Role
- Interactive Button
- Dynamic Message using JavaScript

The project was intentionally kept simple so the primary focus remained on mastering Git and GitHub.

# 🎯 Git & GitHub Skills Practiced

During this project I practiced the following Git commands and concepts:

### Repository Management

- Initialize Git repository
- Check repository status
- Stage files
- Commit changes
- View commit history

### Branching

- Create branches
- Switch branches
- Merge branches
- Delete merged branches

### Tracking Changes

- View file differences
- Restore modified files
- Unstage files
- Push updates to GitHub

### Remote Repository

- Create GitHub repository
- Connect local repository with remote
- Push commits
- Update existing repository

# 📚 Git Commands Practiced

```bash
git init
git status
git add .
git commit -m "Initial commit"

git diff
git log

git restore
git restore --staged

git branch feature-ui
git checkout feature-ui

git merge feature-ui
git branch -d feature-ui

git remote add origin <repository-url>

git push
git push -u origin main
```
